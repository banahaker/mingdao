# 製作動機
學校的原登入介面老舊且沒有響應式設計(RWD)，使學生看了非常的不舒服，不僅如此，學校的網站有時連線速度極慢，此網站也可做為站時替代品使用，因為以上原因我便自行製作一個精美完善的登入畫面

4. 他是真的可以登入的呦!!(暫只支援學生登入)
